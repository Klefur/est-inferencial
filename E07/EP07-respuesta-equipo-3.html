Pregunta 1

Observando los datos, la memorista sospecha que hay diferencias significativas en el tiempo de ejecucin entre las versiones A y C del algoritmo cuando las instancias tienen 60 o ms nodos. Los datos respaldan la intuicin de la memorista? Para responder, filtren los datos para tener las instancias con 60 o ms nodos y seleccionen las columnas de los tiempos de ejecucin de las versiones A y C en formato ancho. Usando como semilla el valor 73, obtenga muestras aleatorias independientes de 24 tiempos registrados por la versin A y 20 tiempos registrados por la versin C del algoritmo. Realicen un anlisis estadstico pertinente (enunciar hiptesis, revisar condiciones, seleccionar prueba) para responder la pregunta planteada, utilizando pruebas no paramtricas de ser necesario.

Hipotesis:

  • H0: No hay diferenencias significativas en el tiempo de ejecucin entre las versiones A y C del algoritmo cuando las instancias tienen 60 o ms nodos.

  • HA: Hay diferenencias significativas en el tiempo de ejecucin entre las versiones A y C del algoritmo cuando las instancias tienen 60 o ms nodos.

set.seed(73)

datos <- read.csv2("EP07 Datos.csv", sep = ",")

instancia <- 1:150
algA <- datos$tiempo.A[datos$n.nodos >= 60]
algC <- datos$tiempo.C[datos$n.nodos >= 60]

tiempos <- data.frame(instancia, algA, algC)

indicesA <- sample(instancia, 24)
indicesC <- sample(as.integer(setdiff(instancia, indicesA)), 20)

muestraA <- tiempos$algA[indicesA]
muestraC <- tiempos$algC[indicesC]


g <- ggqqplot(muestraA)
print(g)

g1 <- ggqqplot(muestraC)
print(g1)

Como se puede apreciar, el grfico de la muestraA, posee valores atpicos, por lo que no se cumple del todo la condicin de la distribucin normal de la prueba t. En consecuencia, se har uso de una prueba no paramtrica con variables numricas. Como se est trabajando con muestras independientes, se utilizar la prueba de suma de rangos de Wilcoxon.

Para trabajar con la prueba de suma de rangos de Wilcoxon, se deben verificar las siguientes condiciones:

1- Las observaciones de ambas muestras son independientes.

2- La escala de medicin empleada debe ser a lo menos ordinal, de modo que tenga sentido hablar de relaciones de orden (igual que, menor que, mayor o igual que).

La primera condicin se cumple, puesto que ambas muestras son independientes entre s.

Trabajando con mediciones de tiempo en milisegundos, podemos afirmar que la escala de medicin es al menos ordinal, ya que los valores se pueden ordenar y es lgico establecer relaciones de orden. Esto se debe a que podemos comparar los valores de tiempo y determinar que, por ejemplo, 1 milisegundo es menor que 100 milisegundos.

Ahora, se procede a trabajar con la prueba de suma de rangos de Wilcoxon.

alfa <- 0.05

# Hacer la prueba de Mann - Whitney .
prueba <- wilcox.test(muestraA,muestraC,alternative = "two.sided" , conf.level = 1 - alfa)
print(prueba)
## 
##  Wilcoxon rank sum exact test
## 
## data:  muestraA and muestraC
## W = 273, p-value = 0.4475
## alternative hypothesis: true location shift is not equal to 0

Como el p-value es mayor al nivel de significancia establecido, se falla al rechazar la hiptesis nula. Por lo tanto, no hay prueba suficiente para respaldar la intuicin de la memorista.
